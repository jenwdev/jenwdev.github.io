Key to CGD Website

Audience: Patients and caregivers of patients with CGD, chronic granulomatous disease.

Project: The Key to CGD website is a dedicated resource designed to empower individuals and families affected by Chronic Granulomatous Disease (CGD). CGD is a rare genetic disorder characterized by a compromised immune system. This website aims to be a comprehensive guide for patients, caregivers, and anyone seeking information about CGD.

Timeframe: 3 months

Technologies: HTML CSS Bootstrap JavaScript CrownPeak CMS
